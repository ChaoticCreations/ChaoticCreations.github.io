Project Description
 Chaotic Creations is a Windows application aimed toward Dungeon Masters of the fifth edition of Dungeons and Dragons. It will include a dice roller visual, a database of random-selection tables, as well as tools for generating non-player characters, encounters, and terrains. The application will be built using Windows Presentation Foundation (WPF) with C# being used for the back end and XAML used for the front end. We will be using XML for database creation and management. The motivation for this project is to help Dungeon Masters be able to improvise during gameplay.
